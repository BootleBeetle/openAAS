Formal Description of Architecture

Creation of a formal description of the management shell architecture and l40-components. Based on the present reference model, a consistent and self-contained generic framework architecture will be identified and formally described. This description will take place parallel in UMD-diagram, in CAEX/AutomationML and as OPC UA Object model in order to make it representable in different technology-illustrations.
